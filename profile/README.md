# Save Cloud

PSVita / Switch / 3ds 掌机第三方存档云备份自制软件。

> Powered by 💖

## 主要功能

- 存档备份与恢复
  > - 本地备份/云端备份
  > - 本地恢复/云端恢复
- 文件管理
  > - 本地文件管理
  > - 本地文件上传
  > - 云端文件管理
  > - 云端文件下载
- 其他功能那请看具体项目主页
